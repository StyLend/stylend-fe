# Stylend Finance

DeFi lending & borrowing protocol frontend built with Next.js.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- wagmi + viem (Web3)
- RainbowKit (Wallet Connect)

## Getting Started

### Prerequisites

- [Bun](https://bun.sh) v1.0+

### Install Dependencies

```bash
bun install
```

### Run Development Server

```bash
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
bun run build
```

### Start Production Server

```bash
bun run start
```

### Lint

```bash
bun run lint
```
